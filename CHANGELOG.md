# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog.

## [1.0.1] - 2026-06-13

### Added

* GitHub Country Request feature
* Premium "No Results Found" state
* Country request callback support
* Improved search experience

### Improved

* Dialog UI and user experience
* Country search performance
* Library documentation and README
* Screenshot section in README

### Fixed

* Country selection dialog issues
* Build and publishing configuration improvements

---

## [1.0.0] - 2026-06-13

### Added

* Country code picker with flag support
* Searchable country list
* Phone number validation using libphonenumber
* Automatic phone number formatting
* Auto country detection using SIM, Network, and Locale
* XML and Kotlin integration support
* Country change listener
* Phone number validity listener
* Multi-language support
* Customizable dialog appearance
* Customizable colors and styles
* Example application module
* JitPack distribution support
